# Dictionary-swift
Lookup Dictionary.app from command line

## Usage
- interpreter ver.
  ```
  ./dict.swift words
  ```
- compiled ver. (needs pre-build)
  ```
  ./dict words
  ```

## Build

```
swiftc dict.swift
```

move `dict` into your `PATH` or alias to it like below.

```
alias dict='path/to/dict'
```
